# Provenance notice

This repository contains an RG-D adaptation of the EXTERNALS inclusive
electron-scattering radiative-correction software.

Original source repository:
https://github.com/utfsm-eg2-data-analysis/EXTERNALS

Historical personal fork used during development:
https://github.com/suman0725/EXTERNALS

The retained Fortran routines derive from that upstream package. RG-D-specific
target definitions, run plans, validation scripts, and correction tables are
maintained in this repository. Consult the upstream repository for its full
history, authorship, and original examples.
